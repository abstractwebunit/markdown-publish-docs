# Поиск

Поиск работает **полностью в браузере** — без сервера и без отправки запросов куда-либо. Индекс ([Pagefind](https://pagefind.app)) собирается вместе с сайтом и грузится по кусочкам, поэтому быстрый даже на больших vault'ах.

## Как пользоваться

- Кликни в поле «Search page or heading…» в сайдбаре — или нажми **Ctrl+K** (`Cmd+K` на Mac).
- Печатай запрос — результаты появляются на лету, с подсветкой совпадений.
- Стрелки ↑/↓ — выбор, **Enter** — открыть, **Esc** — закрыть.

Попробуй прямо сейчас: нажми `Ctrl+K` и набери «конфигурация».

## Что под капотом

При сборке каждая страница индексируется, индекс кладётся рядом со статикой. В браузере подгружаются только нужные фрагменты индекса — поэтому первый поиск занимает миллисекунды, а не качает мегабайты.

Заметки, скрытые режимом `public` ([[Конфигурация]]), в индекс не попадают.

Кроме человеческого поиска у сайта есть и машинный — для AI-агентов: [[AI-агенты (WebMCP)]].
