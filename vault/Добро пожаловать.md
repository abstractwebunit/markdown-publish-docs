# Добро пожаловать

**markdown-publish** превращает папку заметок (Obsidian-vault или просто Markdown) в быстрый статический сайт — вот такой, как этот. Этот сайт собран им же из обычных `.md`-файлов.

> [!tip] Хочешь такой же сайт за пару кликов?
> Открой [[За пару кликов]] — там три кнопки, дальше всё само.

## Что умеет

- **[[Поиск]]** — мгновенный, работает прямо в браузере (нажми `Ctrl+K`).
- **[[Граф связей]]** — интерактивная карта заметок, как в Obsidian (ссылка «Graph view» слева).
- **[[Canvas-доски]]** — `.canvas`-файлы рендерятся как интерактивные доски: вот [[Карта гайдов.canvas|живой пример]].
- **[[AI-агенты (WebMCP)|AI-агенты]]** — сайт отдаёт заметки AI-агентам через WebMCP.
- `[[wikilinks]]`, обычные markdown-ссылки, embeds, callouts, теги, сноски, подсветка кода.
- Тёмная/светлая тема, мобильная версия, превью заметки при наведении на ссылку.
- SEO из коробки: sitemap, robots.txt, llms.txt, OG-карточки, JSON-LD.
- Кириллица в адресах страниц — как у этого сайта.

## Как опубликовать свои заметки

| Путь | Сложность | Гайд |
|---|---|---|
| Кнопка Netlify / Vercel | 2 клика | [[За пару кликов]] |
| Шаблон + GitHub Pages | 3–4 клика | [[GitHub Pages]] |
| Свой репозиторий + workflow | для тех, кто знает git | [[GitHub Pages]] |
| Cloudflare Pages | пара минут | [[Cloudflare Pages]] |
| Локально через CLI | терминал | [[CLI]] |

Опубликовал? Дальше — [[Как обновлять сайт]]. Все настройки (название, язык, какие заметки публиковать) — в [[Конфигурация]].
