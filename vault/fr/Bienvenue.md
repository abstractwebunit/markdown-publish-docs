# Bienvenue

**markdown-publish** transforme un dossier de notes (un vault Obsidian ou simplement du Markdown) en un site statique rapide — exactement comme celui-ci. Ce site a d'ailleurs été généré avec lui, à partir de simples fichiers `.md`.

## Ce qu'il sait faire

- **[[Recherche]]** — instantanée, directement dans le navigateur (appuie sur `Ctrl+K`).
- **[[Graphe de liens]]** — une carte interactive de tes notes, comme dans Obsidian (lien « Graph view » à gauche).
- **[[Tableaux Canvas]]** — les fichiers `.canvas` sont rendus comme des tableaux interactifs : voici un [[Carte des guides.canvas|exemple vivant]].
- **[[Agents IA (WebMCP)|Agents IA]]** — le site sert les notes aux agents IA via WebMCP.
- `[[wikilinks]]`, liens markdown classiques, embeds, callouts, tags, notes de bas de page, coloration syntaxique du code.
- Thème sombre/clair, version mobile, aperçu d'une note au survol d'un lien.
- SEO prêt à l'emploi : sitemap, robots.txt, llms.txt, cartes OG, JSON-LD.
- Caractères non latins et accents dans les adresses de pages — comme sur ce site.
