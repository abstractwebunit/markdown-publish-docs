# Canvas 白板

`.canvas` 文件(Obsidian 使用的 [JSON Canvas](https://jsoncanvas.org) 格式)会渲染成交互式白板。

在线示例:[[指南地图.canvas|指南地图]]——它就以普通 `.canvas` 文件的形式放在这个 vault 里。

## 支持什么

- **文本卡片**——内部的 markdown 会完整渲染。
- **笔记卡片**——显示笔记的内容;点击即可打开。
- **链接卡片**——带标题的外部链接。
- **分组**——带标签的边框。
- 卡片之间的**箭头**,可以带文字说明。

## 交互

- 滚轮缩放,拖动背景平移;打开时白板会自动适配屏幕。
- 卡片可以**拖动**(箭头会跟着走),也可以拖右边/下边或角落来**调整大小**。
- 重新排布只保留到页面刷新——网站是静态的,源 `.canvas` 文件不会被改动。

白板会像普通页面一样出现在左侧导航里。
